# 🌿 Main Branch – Book My Stay App

## 📌 About This Branch

The **main branch** contains the stable and incremental implementation of the  
**Hotel Booking Management System (Book My Stay App)**.

This project is developed use case by use case to demonstrate:

- Core Java fundamentals
- Object-Oriented Programming (OOP)
- Data Structures in real-world scenarios
- Scalable and maintainable system design

Each use case builds upon the previous one to gradually introduce new concepts.

---

## 🏗 Current Implementations in Main Branch

### ✅ Use Case 1 – Application Entry & Welcome Message
- Java program entry point
- `main()` method execution
- Console output
- Basic application startup flow

### ✅ Use Case 2 – Basic Room Types & Static Availability
- Abstract class (`Room`)
- Inheritance (`SingleRoom`, `DoubleRoom`, `SuiteRoom`)
- Polymorphism
- Static availability using variables

### ✅ Use Case 3 – Centralized Room Inventory Management
- Introduction of `HashMap`
- Centralized availability storage
- Encapsulation of inventory logic
- Single source of truth
- O(1) average lookup and update

---

## 🎯 Purpose of Main Branch

The main branch always contains:

- Stable code
- Fully working implementations
- Clean and structured design
- Proper version progression (1.0 → 2.0 → 3.0 → ...)

Experimental features or incomplete work should be developed in separate branches.

---

## 🚀 How to Run

Compile and run the required use case file:

```bash

```

## 📌 Versioning Strategy

- 1.x → Basic system initialization
- 2.x → Object modeling and domain design
- 3.x → Inventory and state management
- Future versions → Booking, search, concurrency, etc.
